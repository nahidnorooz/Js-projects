# Color Generator

A small web app that generates random colors on the screen.

## Features 
- Generates a new random color with a button click.
- Display the color code (HEX/RGB) dynamically.
- live preview of the color.

## Tech Used 
- HTML
- CSS
- JavaScript

## How to run
1. Open the `index.html` file in a browser.
2. Click the "Generate Color" button to see a new color!
