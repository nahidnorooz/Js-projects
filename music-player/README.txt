# Muisc Player

A web-based music player built with JavaScript that provides interactive audio playback and media controls directly in the browser.

## Features
- Play and pause audio tracks
- Skip forward and backward by 10 seconds
- next and previous track navigation
- Dynamic track handling
- clean and responsive interface

## Tech Used
- HTML
- CSS
- JavaScript

## How to run
1. Open `index.html` file in a browser.
2. Use the player controls to navigate through tracks and control playback.
